Data Cleansing Pipeline

Data used was published through the MTA's Open Data Program. In particular, the "MTA Subway Hourly Ridership: Beginning February 2022" was used. As the dataset was very large, python's requests library was used to communicate with the API and obtain data for one day: March 1st, 2023. The data was saved as a CSV. The CSV was then opened using pandas, reformatted so it would be easier to work with and saved in JSON format. In particular, the resulting JSON object containsnested dictionaries of station and time.
